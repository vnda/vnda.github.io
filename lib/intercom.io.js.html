 ¶Como Usar o Liquid?

             

              

              
Existem dois tipos de marcação no líquido:


              
Tags de Lógica: são declarações condicionais. Eles são a afirmação de que diz o código e o que fazer.


              
Tags de saída: são as etiquetas de saída exibir o líquido encontrado entre ele.
                  Você também pode usar filtros para alterar a saída retornado.


              
Para usar o líquido, você vai precisar de recorrer a uma combinação de lógica e de saída de etiquetas usando operadores líquidos. Acções com liquidez muitos dos mesmos operadores como outras linguagens de programação:

              

              

              
Template ler sobre.

            

          

        		
        







1 {% assign order = "false" %}
2 {% orders client_id:client.id %}
3  {% assign order = "true" %}
4  [tr]
5   [td]{{ order.code }}[/td]
6   [td]{{ order.received_at }}[/td]
7   [td]{{ order | order_status }}[/td]
8   [td]{{ order.total | money_format }}[/td]
9   [/tr]
10 {% endorders %}
11 {% if order == "false" %}
12  [tr]
13   [td colspan="3"]Você ainda não tem pedidos[/td]
14  [/tr]
15 {% endif %}

 ¶  {% Menus %}

             

              

              
Listando Menus

              

                 Para listar os menus criados pelo admin da loja deve ser informado o parâmetro "position".
                * loop para exibir os itens de um menu
              

              

              
            

             

              

              
Listando Sub Menus

              

                * menu com submenus (html opcional)
              

              

              
            

          

        		
        







{% menus position:"nome_da_posicao_do_menu_no_admin" %}
  {{ menu.url }} {{ menu.title }}
{% endmenus %}







{% menus position:"nome_da_posicao_do_menu_no_admin" %}
  {% if forloop.first %}<ul>{% endif %}
    <li>
      {% menus parent_id:menu.id %}
        {% if forloop.first %}<ul>{% endif %}
        <li>{{ menu.url }} {{ menu.title }}</li>
        {% if forloop.last %}</ul>{% endif %}
      {% endmenus %}
    </li>
  {% if forloop.last %}</ul>{% endif %}
{% endmenus %}








          		
      

  
        

        

            
 ¶  {% banners %}

             

              

              
Listando Banners

              

                  Para listar os banners criados pelo admin da loja deve ser informado o parâmetro "tag".
                  * loop para exibir os banners
              

              

              
            

          

        		
        








{% banners tag:"nome_da_posicao_no_admin" %}
  {{ banner.title }}
  {{ banner.subtitle }}
  {{ banner.description }}
  {{ banner.file_url | resize: '990x' }}
  {{ banner.url }}
  {% if banner.external %} target="_blank"{% endif %}
{% endbanners %}
* aceita o parâmetro “limit:3”

 ¶ {% tags %}

             

              

              
{% Tags %} Para relacionar um grupo de produtos é preciso utilizar tags e parâmetros.


              * Só é possível agrupar produtos utilizando {% tags %}

              * aceita o parâmetro "limit, page, per_page, type, product_id
              

              

              
Como Usar ler sobre.

            

          

        		
        
          







1  {% tags product_id:product.id type:"marca" limit:1 %}
2    {{ tag.name}}
3    {{ tag.type}}
4    {{ tag.image_url }}
5  {% endtags %}
