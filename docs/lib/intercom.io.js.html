¶ Design

Por Padrão os .psd ou .png deve ser enviados de acordo com a nomenclatura abaixo:

  • home.psd

    tela inicial do site, pode exibir produtos, banners, entre outras coisas. Esta tela deve considerar, caso exista, uma estrutura de header e footer utilizada em toco o canal.

  • tag.psd

    tela de "categoria", exibe uma lista de produtos que estão vinculados a tag. Essa página também pode servir para exibir "sub-categorias".

  • product.psd

    tela que exibe os detalhes dos produtos. Aqui ele pode selecionar as variantes e adicionar a variante selecionada ao carrinho.
